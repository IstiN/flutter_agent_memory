flutter_agent_memory

Dart library for building an LLM-powered knowledge base of questions, answers, and notes — stored as Obsidian-compatible Markdown.

Open live demo GitHub

📚 Structured memory

Capture questions, answers, and notes with tags, areas, topics, and typed relations.

🔍 Search

Search by tags or use natural-language queries with LLM-generated tags and ranking.

🕸️ Graph view

Automatically generate Obsidian-compatible GRAPH.md with Mermaid diagrams.

🔌 Pluggable storage

File system, SQLite, in-memory, HTTP, or browser localStorage — all through one interface.

Try it in your browser

The live demo stores everything in your browser's localStorage. Bring your own OpenAI, OpenRouter, or Ollama token to generate tags and run semantic search — no data leaves your machine except the LLM calls you make.
